Dune Legacy 0.98.6 Released

October 22, 2025

I am thrilled to announce the release of Dune Legacy version 0.98.6 for Windows and macOS! This is a major update bringing significant improvements to AI, multiplayer gameplay, and performance for large-scale battles.

Major Features in 0.98.6

  • QuantBot AI Overhaul: The AI is now significantly more challenging and strategic
    • Difficulty-scaled ornithopter attack strategies (Easy: 1, Medium: 2, Hard: 3, Brutal: 4+ ornithopters before attacking)
    • Intelligent rocket turret defense (builds 2 turrets per enemy ornithopter)
    • Smarter ornithopter targeting - prioritizes rocket turrets and adjusts tactics based on enemy defenses
    • Improved build order logic with dynamic harvester limits and progressive tech requirements
  • Multiplayer LAN Fixes: Enjoy stable multiplayer gameplay
    • Fixed freeze when players pause/unpause the game
    • Added pause state notifications - see which players are paused
    • Optimized network synchronization for smoother gameplay
    • Frame time set to 32ms (~31 FPS) for optimal multiplayer stability
  • Major Performance Improvements: Massive battles run smoothly
    • Performance logging system to track AI, pathfinding, and rendering
    • Optimized pathfinding budget (12ms per frame with intelligent queue system)
    • Fixed pathfinding death spiral that caused frame drops
    • Efficient spatial grid for unit targeting
  • Crash Reporting System: Automatic crash logs with stack traces help diagnose issues
  • Critical Bug Fixes:
    • Fixed 4 null pointer crashes in QuantBot AI
    • Fixed tile squash iterator invalidation crash
    • Fixed carryall repair yard crash
    • Fixed unit targeting and range calculation issues
