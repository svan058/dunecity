Dune Legacy 0.99 Released

November 30, 2025

v0.99 is now out! This major release brings a complete mod system, full AI rebalancing, and tons of gameplay improvements.

🎮 Main Changes

  • God Mode: Type muaddib during single-player for immortality - great for play testing!
  • Mod System: Mods can now be created and loaded for play in single and multiplayer! Files live in your app data folder in their own mod folder.
  • Full QuantBot Rebalance: Completely rebalanced AI for the standard campaign - let me know how you find it!
  • Harvester Limits: Implemented harvester limits - 40 for 128x128 maps and bigger, scaling down for smaller maps. You can still override it in settings.
  • Mod Editor: Edit default AI (QuantBot only), units, buildings, and game settings to create new default values. For example, you could balance troopers and infantry to make them useful, or tweak anything else!

📦 Complete Mod System

  • Mod Editor: Create and edit mods directly in-game with custom names and version tracking
  • Mod Management Menu: Access via Main Menu → MODS to list, activate, create, edit, and delete mods
  • Multiplayer Mod Sync: Host's mod automatically synced to all clients - no manual file sharing required!
  • Metaserver Integration: Server browser shows mod name and version for each game
  • Moddable Files: ObjectData.ini (unit stats), GameOptions.ini (game rules), QuantBot Config.ini (AI behavior)

⚔️ Major Combat & Unit Behavior Fixes

  • Instant Counter-Attack: Units now immediately respond when damaged (no more 2-second delay)
  • Pursuit Behavior Fixed: Units now move to exact weapon range instead of stopping 3 tiles short
  • Saboteur Fixes: AI palace-spawned saboteurs now work correctly with proper pathfinding to occupied tiles
  • AMBUSH Mode: Infantry in AMBUSH mode now pursue attackers (matches Dynasty behavior)
  • Sandworm Improvements: Sandworms counter-attack when damaged from range

🤖 AI Improvements

  • Rebalanced Campaign Difficulty: Better progression from Easy to Brutal with improved spice management
  • Factory Priority System: Light Factory/WOR only produce when no Heavy Factory exists (fixes trike spam)
  • Enhanced Kiting: Launchers and Deviators kite on all difficulties with proportional retreat distances
  • Support Mode Awareness: Long-range units no longer kite when in support mode

🎯 Bug Fixes

  • Harvester Recovery: Automatic stuck detection and retry system prevents infinite pathfinding loops
  • Deviated Unit Positioning: Deviated units now properly move to squad center with tighter 2-tile radius
  • Deathhand Accuracy: Dynasty scatter algorithm implemented for more predictable strikes
  • Carryall Improvements: Distance-based damage falloff and better pickup/dropoff logic

🎮 Cheat Codes & God Mode

  • New God Mode: Type "muaddib" during single-player to toggle immortality for human units/structures
  • Game Options Menu: Checkbox for "Immortal Human Player" when starting new games
  • Special Interactions: Sandworms choke on immortal units, safe carryall drops, survives capture attempts

🗺️ New Multiplayer Maps

  • 4P Moshpit with Garbages (128x128)
  • 6P Alkozeltser 2 (128x128)
  • 6P Alkozeltser 4 (256x256)
  • 6P Full Wormage (128x128)

🌐 Multiplayer Improvements

  • Automatic Map Sharing: Custom maps automatically saved to clients when joining games
  • Automatic Mod Sync: Host's mod files transferred and synced to all clients with checksum verification
  • Server Browser: Shows mod name and version for each active game

⚙️ Performance & Technical

  • Proper VSync: Implemented using SDL_RenderSetVSync() with multi-tier FPS thresholds
  • Configuration Standardization: Renamed .bak files to .ini.default for clarity
  • macOS App Bundle Fix: Fixed data directory path resolution for proper mod seeding
