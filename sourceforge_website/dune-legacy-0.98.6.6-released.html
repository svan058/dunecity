Dune Legacy 0.98.6.6 Released

November 3, 2025

Dune Legacy 0.98.6.6 is now available! This update focuses on multiplayer stability and removes the last remaining synchronization issues reported since 0.98.6 shipped.

What's New in 0.98.6.6

🛰️ Multiplayer Stability

  • Network pacing restored: Multiplayer games no longer hang on "Waiting for other players"
  • Command synchronization: Improved buffering keeps all peers aligned even under packet loss
  • Pause support: Players can now pause and resume matches with synchronized notifications

🧭 Deterministic Pathfinding

  • Token-based budgets: Replaces time-sliced pathfinding with per-frame node limits so every client processes the exact same work
  • Consistent performance: Removes CPU-speed dependent divergence and keeps large battles smooth

✈️ Ornithopter Improvements

  • Original targeting restored: Attack priority matches classic Dune II behavior
  • Quick select button: New UI control selects all ornithopters instantly
