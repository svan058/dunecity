Dune Legacy 0.98.6.2 Released

October 24, 2025

Dune Legacy 0.98.6.2 is now available! This release focuses on game balance and adds user-editable configuration files for units, structures, and AI behavior.

What's New in 0.98.6.2

🎛️ User-Editable Configuration Files

  • ObjectData.ini: Customize unit/structure stats (HP, speed, damage, cost, build time) without recompiling
  • QuantBot Config.ini: Adjust AI difficulty, unit composition, attack behavior, and harvester limits
  • Config files auto-copied to user directory on first run (Windows: %APPDATA%\dunelegacy\config\)
  • Includes detailed inline documentation and examples

⚖️ Ornithopter Balance Changes

  • Speed: 22 → 15 (32% slower, turret rockets can now catch them)
  • Attack waves: Hard (2→1 ornis), Brutal (4→3 ornis) - smaller, more frequent raids
  • AI builds ornithopters but won't attack with them on Easy/Medium difficulties

🚀 Rocket Turret Improvements

  • Scan frequency: 3 seconds → 0.2 seconds (15x faster targeting)
  • Fixed close-range air targeting bug (turrets now shoot air units at all ranges)
  • Rocket detonation timer: 30 → 60 cycles (doubled pursuit time)
  • Immediate retaliation when damaged

🤖 AI Defense Improvements

  • Builds 4 insurance rocket turrets proactively (before seeing enemy ornithopters)
  • Proactive CY upgrade to level 2 after Starport (no more reactive delays)
  • Counter logic fixed: builds 2 turrets per enemy ornithopter (max from single house, not sum)
  • Prerequisite checking: ensures Windtrap + Radar before building turrets
