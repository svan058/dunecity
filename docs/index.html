Dune Legacy

Original engine

DuneCity is a fork of Dune Legacy, an open-source recreation of Dune II. For the classic experience without city-building, grab Dune Legacy from SourceForge.

  • SourceForge

Requirements

Dune II data files required. DuneCity needs the original Dune II PAK files to run (DUNE.PAK, SCENARIO.PAK, etc.). You must legally own a copy of Dune II: The Building of a Dynasty. The game will prompt you to locate these files on first launch.
